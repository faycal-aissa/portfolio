📡 Infrastructure VLAN & Routage

Conception d’une architecture réseau segmentée avec configuration du routage inter-VLAN. Mise en place de protocoles de redondance de passerelle (HSRP/VRRP) et de routage dynamique (OSPF) pour assurer la tolérance aux pannes.

⚖️ Haute Disponibilité & Load Balancing Web

Déploiement d'un cluster Linux garantissant la continuité de service. Configuration de Corosync et Pacemaker pour le basculement d'IP virtuelle (Failover), répartition de charge via HAProxy, et réplication de bases de données MySQL en architecture Maître/Esclave.

📄 Voir le dossier

🏢 Administration Active Directory & GPO

Déploiement et sécurisation d'un environnement Microsoft Windows Server. Création d'une arborescence Active Directory complète, gestion des droits d'accès aux serveurs de fichiers (méthode AGDLP) et automatisation du parc via le déploiement de stratégies de groupe (GPO).

📄 Voir la configuration

📊 Supervision Réseau & Système (Centreon)

Mise en place d'une solution de supervision proactive de l'infrastructure via Centreon. Configuration des remontées d'alertes via le protocole SNMP pour monitorer la santé des serveurs (CPU, RAM, disques) et l'état des équipements d'interconnexion.
